# PIFBot 硬件系统分析

## 1. 硬件架构概览

PIFBot 硬件系统采用模块化设计，主要包含以下子系统：

```
┌────────────────────────────────────────────────────────────┐
│                    PIFBot 硬件架构                          │
├────────────────────────────────────────────────────────────┤
│                                                            │
│  ┌──────────────┐    ┌──────────────┐    ┌─────────────┐  │
│  │   控制系统    │    │   感知系统    │    │  电源系统   │  │
│  │  OpenSTM32   │    │  传感器模块   │    │  供电管理   │  │
│  └──────────────┘    └──────────────┘    └─────────────┘  │
│                                                            │
│  ┌──────────────┐    ┌──────────────┐    ┌─────────────┐  │
│  │   驱动系统    │    │   计算系统    │    │  机械系统   │  │
│  │  电机+驱动   │    │ Raspberry Pi  │    │  底盘结构   │  │
│  └──────────────┘    └──────────────┘    └─────────────┘  │
│                                                            │
└────────────────────────────────────────────────────────────┘
```

## 2. 主控制器板 - OpenSTM32

### 2.1 核心规格

| 参数 | 规格 |
|-----|------|
| 主控芯片 | STM32F4 系列 |
| 内核 | ARM Cortex-M4 |
| FPU | 单精度浮点运算单元 |
| 时钟频率 | 最高 168 MHz |
| 封装 | LQFP64 |

### 2.2 功能特性

| 功能模块 | 描述 |
|---------|------|
| GPIO | 通用数字输入输出 |
| ADC | 模拟信号采集 |
| SPI | 串行外设接口 |
| I2C | 两线式串行总线 |
| UART | 通用异步收发器 |
| I2S | 集成音频接口 |
| IMU | 板载 MPU9250 (9-DOF) |

### 2.3 引脚分配

```
┌─────────────────────────────────────────────────────┐
│                  OpenSTM32F4 LQFP64                  │
├─────────────────────────────────────────────────────┤
│                                                     │
│  电机控制接口:                                       │
│  ├── PWM 输出 → TB6560 步进电机驱动器               │
│  ├── 方向控制信号                                   │
│  └── 使能信号                                       │
│                                                     │
│  编码器接口:                                        │
│  ├── 左轮编码器 A/B 相                              │
│  └── 右轮编码器 A/B 相                              │
│                                                     │
│  通信接口:                                          │
│  ├── UART → Raspberry Pi (rosserial)               │
│  ├── I2C → 外部传感器扩展                           │
│  └── SPI → 高速外设                                 │
│                                                     │
│  传感器接口:                                        │
│  └── I2C → MPU9250 IMU                              │
│                                                     │
└─────────────────────────────────────────────────────┘
```

## 3. 传感器系统

### 3.1 RPLIDAR A1 激光雷达

| 参数 | 规格 |
|-----|------|
| 扫描角度 | 360° |
| 扫描频率 | 5.5 Hz |
| 测距范围 | 0.15m - 12m |
| 测距精度 | ~0.5mm |
| 接口 | USB |
| 角度分辨率 | 1° |

**应用场景**:
- 2D SLAM 建图
- 障碍物检测
- 导航避障

### 3.2 Intel RealSense D415

| 参数 | 规格 |
|-----|------|
| 深度技术 | 结构光 (RealSense) |
| 深度范围 | 0.3m - 10m |
| 分辨率 | 最高 1920×1080 |
| 帧率 | 最高 90 fps |
| 接口 | USB 3.0 |
| 视场角 | ~69° × 42° |

**应用场景**:
- 3D SLAM (RTAB-Map)
- 物体识别
- 场景理解
- 手势识别

### 3.3 MPU9250 IMU

| 参数 | 规格 |
|-----|------|
| 加速度计 | 3轴, ±2/4/8/16g |
| 陀螺仪 | 3轴, ±250/500/1000/2000°/s |
| 磁力计 | 3轴, ±4800µT |
| 接口 | I2C/SPI |
| 自由度 | 9-DOF |

**应用场景**:
- 姿态估计
- 运动融合
- 里程计辅助

## 4. 驱动系统

### 4.1 步进电机

| 参数 | 规格 |
|-----|------|
| 型号 | 42×42mm 步进电机 |
| 轮径 | 65mm |
| 步距角 | 1.8° (典型值) |
| 驱动方式 | 双相 |

### 4.2 TB6560 驱动器

| 参数 | 规格 |
|-----|------|
| 输入电压 | 12V |
| 输出电流 | 可调 (最大 3A) |
| 细分 | 1/1, 1/2, 1/8, 1/16 |
| 控制信号 | 步进脉冲 + 方向 |

### 4.3 差速驱动模型

```
        前进方向
            ↑
      ┌─────────┐
      │    ○    │ ← 俯视图
      │  ┌───┐  │
   ←──┤  │   │  ├──→ 右轮
      │  └───┘  │
      │  ┌───┐  │
      │  │   │  │
      │  └───┘  │
      │    ●    │ ← 万向轮
      └─────────┘
         左轮

运动学方程:
  v = (v_left + v_right) / 2
  ω = (v_right - v_left) / L
  
  其中:
    v: 线速度
    ω: 角速度
    L: 轴距 (165mm)
```

## 7. 接线与连接

### 7.1 主要连接

| 连接 | 类型 | 说明 |
