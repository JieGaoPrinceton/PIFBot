# STM32 核心系统技术文档

## 1. 概述

本文档以 STM32F405RGT6 为核心，详细说明 PIFBot 项目中与 STM32 相关的硬件接口和软件架构。

### 1.1 STM32 核心芯片规格

| 参数 | 规格 |
|-----|------|
| 型号 | STM32F405RGT6 |
| 架构 | ARM Cortex-M4 |
| 主频 | 168 MHz |
| FPU | 单精度浮点运算单元 |
| Flash | 1 MB |
| SRAM | 192 KB |
| 封装 | LQFP64 |

### 1.2 开发环境

| 工具 | 说明 |
|-----|------|
| IDE | STM32CubeIDE |
| 配置工具 | STM32CubeMX |
| HAL 库 | STM32F4xx HAL Driver |
| 调试接口 | SWD/JTAG |

### 3.3 核心模块详解

#### 3.3.1 主控制流程

```cpp
// main.cpp - 主程序入口
int main(void)
{
    // 1. HAL 库初始化
    HAL_Init();
    
    // 2. 系统时钟配置 (168 MHz)
    SystemClock_Config();
    
    // 3. 外设初始化
    MX_GPIO_Init();      // GPIO
    MX_DMA_Init();       // DMA
    MX_I2C2_Init();      // I2C (IMU)
    MX_TIM2_Init();      // 定时器2 (左编码器)
    MX_TIM3_Init();      // 定时器3 (右编码器)
    MX_TIM13_Init();     // 定时器13 (左电机PWM)
    MX_TIM14_Init();     // 定时器14 (右电机PWM)
    MX_UART4_Init();     // UART4 (rosserial)
    MX_USART3_UART_Init(); // USART3 (调试日志)
    
    // 4. 底盘控制初始化
    BaseControl_Init();
    
    // 5. 主循环
    while (1)
    {
        BaseControl_Main();
    }
}
```

#### 3.3.2 BaseControl 主循环

```cpp
// BaseControl.c - 主循环逻辑
void BaseControl_Main(void)
{
    uint32_t t = hw_intf_get_time_ms();
    
    // 1. 更新 ROS 时间
    base_control_update_time();
    
    // 2. 电机速度控制 (10 Hz)
    if ((t - base_control_time_update[0]) >= 100)
    {
        base_control_update_goal_vel();
        
        // 超时保护 - 无命令时停止电机
        if ((t - base_control_get_time_callback_cmdvel()) > 500)
        {
            base_control_set_zero_vel();  // 停止
        }
        else
        {
            base_control_set_goal_vel();  // 执行目标速度
        }
        base_control_time_update[0] = t;
    }
    
    // 3. 发布速度信息 (5 Hz)
    if ((t - base_control_time_update[1]) >= 200)
    {
        base_control_publish_cmdvel_from_motor_msg();
        base_control_time_update[1] = t;
    }
    
    // 4. 发布里程计/TF/JointState (5 Hz)
    if ((t - base_control_time_update[2]) >= 200)
    {
        base_control_publish_drive_info();
        base_control_time_update[2] = t;
    }
    
    // 5. 更新 IMU 数据 (1000 Hz)
    if ((t - base_control_time_update[4]) >= 1)
    {
        base_control_update_imu();
        base_control_time_update[4] = t;
    }
    
    // 6. 发布 IMU 消息 (15 Hz)
    if ((t - base_control_time_update[3]) >= 67)
    {
        base_control_publish_imu_msg();
        base_control_time_update[3] = t;
    }
    
    // 7. ROS 节点同步
    base_control_spin_once();
}
```

### 4.3 PWM 电机控制

```c
// 电机 PWM 参数计算
stepmotor_status_t hw_intf_leftmotor_set_pwm_freq(uint32_t freq)
{
    if (freq == 0)
    {
        // 停止输出
        __HAL_TIM_SET_AUTORELOAD(&htim13, 0);
        __HAL_TIM_SET_PRESCALER(&htim13, 0);
        __HAL_TIM_SET_COMPARE(&htim13, TIM_CHANNEL_1, 0);
        return STEPMOTOR_STATUS_SUCCESS;
    }
    
    // 计算定时器参数
    uint32_t apb_freq = 84000000;  // APB 时钟频率
    uint32_t conduct = apb_freq / freq;
    uint16_t prescaler = conduct / 65535 + 1;
    uint16_t period = (uint16_t)(conduct / (prescaler + 1)) - 1;
    
    // 设置定时器
    __HAL_TIM_SET_AUTORELOAD(&htim13, period);
    __HAL_TIM_SET_PRESCALER(&htim13, prescaler);
    
    return STEPMOTOR_STATUS_SUCCESS;
}
```

## 5. ROS 通信接口

### 5.1 rosserial 协议

STM32 通过 rosserial 协议与 ROS 系统通信，使用 UART4 串口。

```
┌─────────────────┐                    ┌─────────────────┐
│   STM32F405     │    UART4 (115200)  │  Raspberry Pi   │
│                 │◄──────────────────►│                 │
│  rosserial node │   rosserial 协议   │  rosserial_py   │
└─────────────────┘                    └─────────────────┘
```

### 5.2 ROS 话题定义

```cpp
// 订阅话题
ros::Subscriber<geometry_msgs::Twist> cmd_vel_sub("cmd_vel", callback);  // 速度命令
ros::Subscriber<std_msgs::Empty> reset_sub("reset", callback);           // 重置里程计

// 发布话题
ros::Publisher imu_pub("imu", &imu_msg);                    // IMU 数据
ros::Publisher odom_pub("odom", &odom);                     // 里程计
ros::Publisher joint_states_pub("joint_states", &joint);   // 关节状态
ros::Publisher cmd_vel_motor_pub("cmd_vel_motor", &vel);   // 电机速度反馈
```

```
    │                                 │
    │    ○                       ○    │  ← 万向轮
    │                                 │
    │  ┌───┐                   ┌───┐  │
    │  │ L │                   │ R │  │  ← 驱动轮
    │  │   │◄─── 165mm ────►   │   │  │
    │  └───┘                   └───┘  │
    │                                 │
    └─────────────────────────────────┘
```

### 6.2 运动学参数

```c
// BaseControl_Define.h
#define WHEEL_RADIUS        0.033    // 轮半径 (m)
#define WHEEL_SEPARATION    0.165    // 轴距 (m)
#define TURNING_RADIUS      0.08     // 转弯半径 (m)
#define ROBOT_RADIUS        0.1      // 机器人半径 (m)

// 步进电机参数
#define MICROSTEP_DIV       8        // 细分
#define NUM_PULSE_PER_ROUND 200      // 每圈脉冲数

// 速度限制
#define MAX_LINEAR_VELOCITY  (WHEEL_RADIUS * 2 * PI * 60 / 60)  // ~0.21 m/s
#define MAX_ANGULAR_VELOCITY (MAX_LINEAR_VELOCITY / TURNING_RADIUS)
```

### 6.3 运动学方程

```cpp
// 正运动学: 轮速 -> 机器人速度
v_linear  = (v_left + v_right) / 2
v_angular = (v_right - v_left) / WHEEL_SEPARATION

// 逆运动学: 机器人速度 -> 轮速
v_left  = v_linear - v_angular * WHEEL_SEPARATION / 2
v_right = v_linear + v_angular * WHEEL_SEPARATION / 2

// 里程计计算
delta_distance = (delta_left + delta_right) / 2
delta_theta = (delta_right - delta_left) / WHEEL_SEPARATION

x += delta_distance * cos(theta + delta_theta/2)
y += delta_distance * sin(theta + delta_theta/2)
theta += delta_theta
```

### 8.1 时钟树

```
                    HSI (16 MHz)
                         │
                         ▼
                    ┌─────────┐
                    │   PLL   │
                    │ M = 8   │
                    │ N = 168 │
                    │ P = 2   │

                    └────┬────┘
                         │
                         ▼
                   SYSCLK = 168 MHz
                         │
         ┌───────────────┼───────────────┐
         │               │               │
         ▼               ▼               ▼
    AHB (168 MHz)  APB1 (42 MHz)  APB2 (84 MHz)
         │               │               │
         ▼               ▼               ▼
    HCLK = 168MHz  TIM2/3 = 84MHz  TIM13/14 = 84MHz
```

### 8.2 时钟配置代码

```c
void SystemClock_Config(void)
{
    // PLL 配置: 168 MHz
    RCC_OscInitStruct.PLL.PLLM = 8;      // HSI/8 = 2MHz
    RCC_OscInitStruct.PLL.PLLN = 168;    // 2MHz × 168 = 336MHz
    RCC_OscInitStruct.PLL.PLLP = 2;      // 336MHz / 2 = 168MHz
    RCC_OscInitStruct.PLL.PLLQ = 4;      // USB/SDIO = 48MHz
    
    // 总线分频
    RCC_ClkInitStruct.AHBCLKDivider  = RCC_SYSCLK_DIV1;   // 168 MHz
    RCC_ClkInitStruct.APB1CLKDivider = RCC_HCLK_DIV4;     // 42 MHz
    RCC_ClkInitStruct.APB2CLKDivider = RCC_HCLK_DIV2;     // 84 MHz
}
```
