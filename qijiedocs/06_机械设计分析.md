# PIFBot 机械设计分析

## 1. 机械系统概述

PIFBot 采用圆形底盘的差速驱动平台设计，具有结构简单、运动灵活的特点。

```
┌────────────────────────────────────────────────────────────┐
│                    PIFBot 机械架构                          │
├────────────────────────────────────────────────────────────┤
│                                                            │
│                     ┌──────────┐                           │
│                     │  Layer 3 │ ← 传感器层                │
│                     │ (顶部)   │   相机/LiDAR              │
│                     └────┬─────┘                           │
│                          │                                 │
│                     ┌────┴─────┐                           │
│                     │  Layer 2 │ ← 计算层                  │
│                     │ (中层)   │   Raspberry Pi            │
│                     └────┬─────┘                           │
│                          │                                 │
│                     ┌────┴─────┐                           │
│                     │  Layer 1 │ ← 控制/电源层             │
│                     │ (底层)   │   OpenSTM32/电池          │
│                     └────┬─────┘                           │
│                          │                                 │
│                    ╔═══════════╗                           │
│                    ║  底盘     ║ ← 主体结构                │
│                    ║  (底盘)   ║   云母板激光切割          │
│                    ╚═════╤═════╝                           │
│                          │                                 │
│          ┌───────────────┼───────────────┐                │
│          │               │               │                │
│     ┌────┴────┐    ┌─────┴─────┐   ┌────┴────┐           │
│     │ 左驱动轮 │    │  万向轮   │   │ 右驱动轮 │           │
│     │ (主动)  │    │  (从动)   │   │ (主动)  │           │
│     └─────────┘    └───────────┘   └─────────┘           │
│                                                            │
└────────────────────────────────────────────────────────────┘
```

## 2. 规格参数

### 2.1 尺寸规格

| 参数 | 数值 | 说明 |
|-----|------|------|
| 底盘直径 | 200 mm | 圆形底盘 |
| 最大高度 | 212 mm | 包含传感器 |
| 轮子直径 | 65 mm | 驱动轮 |
| 轴距 | 165 mm | 两驱动轮中心距 |
| 底盘材料 | 云母板 | 激光切割 |

### 2.2 质量估算

| 组件 | 估算质量 |
|-----|---------|
| 底盘 + 层板 | ~1.5 kg |
| 电机 + 驱动器 | ~0.5 kg |
| Raspberry Pi + 配件 | ~0.2 kg |
| OpenSTM32 + 传感器 | ~0.3 kg |
| 电池 (3S LiPo) | ~0.4 kg |
| **总计** | **~2.9 kg** |

## 5. 底盘设计

### 5.1 圆形底盘

```
                    前进方向
                       ↑
                       
            ┌──────────────────┐
          ╱                        ╲
        ╱                            ╲
       │                              │
      │      ┌────────────────┐       │
      │      │                │       │
      │      │   中心区域     │       │
      │      │                │       │
      │      └────────────────┘       │
       │                              │
        ╲                            ╱
          ╲                        ╱
            └──────────────────┘
                 ║          ║
                 ║          ║
              左轮        右轮
            (驱动)      (驱动)
            
        ← 165mm (轴距) →
        
        ╔══════════════════╗
        ║  Φ = 200mm       ║
        ║  (底盘直径)      ║
        ╚══════════════════╝
```

### 5.2 设计优势

| 优势 | 说明 |
|-----|------|
| 各向同性 | 圆形底盘可原地旋转，运动灵活 |
| 结构简单 | 减少设计复杂度 |
| 空间利用 | 圆形布局最大化内部空间 |
| 避障友好 | 圆形轮廓便于碰撞检测 |

## 6. 驱动系统设计

### 6.1 差速驱动配置

```
                前进方向 (↑)
                    
    ┌─────────────────────────────────┐
    │                                 │
    │    ○                       ○    │  ← 万向轮 (后)
    │                                 │
    │                                 │
    │                                 │
    │  ┌───┐                   ┌───┐  │
    │  │ L │                   │ R │  │  ← 驱动轮
    │  │   │◄─── 165mm ────►   │   │  │
    │  └───┘                   └───┘  │
    │   ▲                         ▲   │
    │   │                         │   │
    │ 步进电机                 步进电机│
    │ + 编码器               + 编码器 │
    │                                 │
    └─────────────────────────────────┘

运动学方程:
  v = (v_L + v_R) / 2
  ω = (v_R - v_L) / L
  
  其中 L = 165mm (轴距)
```

### 6.2 轮子规格

| 参数 | 数值 |
|-----|------|
| 轮径 | 65 mm |
| 轮宽 | ~25 mm (典型) |
| 类型 | 固定驱动轮 |
| 材料 | 橡胶/硅胶 |

### 8.2 制造建议

```yaml
激光切割参数建议:
  材料: 云母板 / 亚克力 (3-5mm)
  功率: 根据材料厚度调整
  速度: 确保切透边缘光滑

  - 去除毛刺
  - 边缘打磨
  - 必要时抛光

  - M3 螺丝/螺母: 若干
  - M3 铜柱: 20mm × 4, 30mm × 4, 50mm × 4
  - 电机安装螺丝: M3 或 M4
```

## 9. 扩展性设计

### 9.1 可扩展方向

| 扩展方向 | 描述 |
|---------|------|
| 传感器扩展 | 预留 I2C/SPI 接口安装额外传感器 |
| 计算扩展 | 可替换为更强大的 SBC (如 Jetson Nano) |
| 机械扩展 | 层板设计支持添加更多层 |
| 电源扩展 | 支持更大容量电池 |

### 9.2 改进建议

1. **增加电池仓设计**: 优化电池固定方式
2. **线缆管理**: 增加线槽或扎带固定点
3. **防护设计**: 可增加外壳防护
4. **模块化**: 各层可独立拆卸维护
