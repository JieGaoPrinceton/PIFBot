# PIFBot 固件系统分析

## 1. 固件架构概述

PIFBot 的嵌入式固件运行在 OpenSTM32 控制板上，负责底层硬件控制和与 ROS 系统的通信。

│                                                            │
│  ┌──────────────────────────────────────────────────────┐  │

│  │  ┌────────────┐ ┌────────────┐ ┌────────────────┐   │  │
│  │  │ 里程计计算  │ │ 运动控制   │ │ 传感器融合     │   │  │
│  │  └────────────┘ └────────────┘ └────────────────┘   │  │
│  └──────────────────────────────────────────────────────┘  │

│  │  ┌────────────┐ ┌────────────┐ ┌────────────────┐   │  │

## 2. 开发框架 - STM-IDF

### 2.1 框架简介

STM-IDF (STM32 Integrated Development Framework) 是专为 STM32 开发的开源框架：

| 特性 | 描述 |
|-----|------|
| 语言 | C/C++ |
| 许可证 | MIT |
| 代码仓库 | github.com/phonght32/stm-idf |
| 目标芯片 | STM32F4 系列 |

### 2.2 框架组件

```yaml

```

## 3. 核心功能模块

### 3.1 里程计模块 (Odometry)

```cpp
// 里程计计算伪代码
class Odometry {
private:
    double x, y, theta;        // 位姿
    double v_left, v_right;    // 轮速
    double wheel_radius;       // 轮半径
    double wheel_base;         // 轴距
    
public:
    void update(int left_ticks, int right_ticks, double dt) {
        // 1. 计算轮子移动距离
        double dist_left = ticks_to_meters(left_ticks);
        double dist_right = ticks_to_meters(right_ticks);
        
        // 2. 计算线速度和角速度
        double v = (dist_left + dist_right) / (2.0 * dt);
        double omega = (dist_right - dist_left) / (wheel_base * dt);
        
        // 3. 更新位姿 (差速驱动运动学)
        double delta_theta = omega * dt;
        double delta_x = v * cos(theta + delta_theta/2) * dt;
        double delta_y = v * sin(theta + delta_theta/2) * dt;
        
        x += delta_x;
        y += delta_y;
        theta += delta_theta;
        
        // 4. 角度归一化
        theta = normalize_angle(theta);
    }
    
    nav_msgs::Odometry get_odom_msg() {
        // 返回 ROS 里程计消息
    }
};
```

### 3.2 运动控制模块

```cpp
// 运动控制器伪代码
class MotionController {
private:
    PIDController pid_left;
    PIDController pid_right;
    double target_v_left;
    double target_v_right;
    
public:
    void set_velocity(double linear, double angular) {
        // 差速驱动逆运动学
        target_v_left = linear - angular * wheel_base / 2.0;
        target_v_right = linear + angular * wheel_base / 2.0;
    }
    
    void update(double current_v_left, double current_v_right) {
        // PID 控制
        double pwm_left = pid_left.compute(target_v_left, current_v_left);
        double pwm_right = pid_right.compute(target_v_right, current_v_right);
        
        // 设置电机 PWM
        motor_set_pwm(LEFT, pwm_left);
        motor_set_pwm(RIGHT, pwm_right);
    }
};
```

### 3.3 rosserial 通信模块

```cpp
// rosserial 接口伪代码
#include <ros.h>
#include <geometry_msgs/Twist.h>
#include <nav_msgs/Odometry.h>
#include <sensor_msgs/Imu.h>
#include <tf/tf.h>

class ROSInterface {
private:
    ros::NodeHandle nh;
    
    // 订阅者
    ros::Subscriber<geometry_msgs::Twist> cmd_vel_sub;
    
    // 发布者
    ros::Publisher odom_pub;
    ros::Publisher imu_pub;
    
    // 消息对象
    nav_msgs::Odometry odom_msg;
    sensor_msgs::Imu imu_msg;
    
public:
    void init() {
        nh.initNode();
        nh.subscribe(cmd_vel_sub);
        nh.advertise(odom_pub);
        nh.advertise(imu_pub);
    }
    
    void cmd_vel_callback(const geometry_msgs::Twist& msg) {
        // 接收速度命令
        double linear = msg.linear.x;
        double angular = msg.angular.z;
        motion_controller.set_velocity(linear, angular);
    }
    
    void spin() {
        // 发布里程计和 IMU 数据
        odom_pub.publish(&odom_msg);
        imu_pub.publish(&imu_msg);
        nh.spinOnce();
    }
};
```

## 4. 控制算法

### 4.1 PID 控制器

```
              ┌─────────────────────────────────────────────┐
              │                  PID 控制器                  │
              │                                             │
target ──(+)──┤ Kp × e + Ki × ∫e dt + Kd × de/dt  ─────┼──► output
              │                                             │
        (─)──┤                                             │
              │                                             │
         current                                           │
              └─────────────────────────────────────────────┘

参数整定建议:
┌──────────┬────────────┬─────────────────────────────────┐
│ 参数     │ 作用       │ 调整建议                         │
├──────────┼────────────┼─────────────────────────────────┤
│ Kp       │ 比例增益   │ 响应速度，过大则震荡             │
│ Ki       │ 积分增益   │ 消除稳态误差，过大则超调         │
│ Kd       │ 微分增益   │ 阻尼作用，抑制震荡               │
└──────────┴────────────┴─────────────────────────────────┘
```

### 4.2 电机控制时序

```
┌─────────────────────────────────────────────────────────┐
│                     主控制循环                           │
│                   (1 kHz 典型值)                         │
├─────────────────────────────────────────────────────────┤
│                                                         │
│  循环周期 (1ms):                                        │
│  │                                                      │
│  ├── 1. 读取编码器计数值                                │
│  │      └── TIMx->CNT                                   │
│  │                                                      │
│  ├── 2. 计算当前轮速                                    │
│  │      └── v = Δticks × scale / Δt                    │
│  │                                                      │
│  ├── 3. PID 计算                                        │
│  │      └── pwm = PID(target_v, current_v)             │
│  │                                                      │
│  ├── 4. 更新 PWM 输出                                   │
│  │      └── TIMx->CCR = pwm                            │
│  │                                                      │
│  ├── 5. 更新里程计                                      │
│  │      └── odom.update(ticks_left, ticks_right, dt)   │
│  │                                                      │
│  ├── 6. 读取 IMU 数据 (每 N 个周期)                     │
│  │      └── I2C 读取 MPU9250                           │
│  │                                                      │
│  └── 7. rosserial 处理 (异步/定时)                      │
│                                                         │
└─────────────────────────────────────────────────────────┘
```

## 5. 传感器驱动

### 5.1 编码器接口

```
编码器信号:
┌─────────┐                    STM32
│ 编码器   │─── A 相 ──────────► TIMx_CH1 (编码器模式)
│         │─── B 相 ──────────► TIMx_CH2
└─────────┘

STM32 定时器配置:
- 编码器接口模式 (Encoder Interface Mode)
- 自动计数方向识别
- 4 倍频计数 (A/B 相上升沿和下降沿都计数)

分辨率计算:
  counts_per_revolution = PPR × 4
  distance_per_count = wheel_circumference / counts_per_revolution
```

### 5.2 MPU9250 IMU 驱动

```cpp
// IMU 数据读取伪代码
class MPU9250 {
private:
    I2C i2c;
    uint8_t address;
    
public:
    struct IMUData {
        float ax, ay, az;    // 加速度
        float gx, gy, gz;    // 陀螺仪
        float mx, my, mz;    // 磁力计
    };
    
    void init() {
        // 初始化传感器
        write_register(PWR_MGMT_1, 0x00);  // 唤醒
        write_register(GYRO_CONFIG, 0x08); // ±500°/s
        write_register(ACCEL_CONFIG, 0x08);// ±4g
    }
    
    IMUData read() {
        IMUData data;
        // 读取加速度和陀螺仪
        uint8_t buffer[14];
        read_burst(ACCEL_XOUT_H, buffer, 14);
        
        data.ax = (int16_t)(buffer[0] << 8 | buffer[1]) / 8192.0;
        data.ay = (int16_t)(buffer[2] << 8 | buffer[3]) / 8192.0;
        data.az = (int16_t)(buffer[4] << 8 | buffer[5]) / 8192.0;
        
        data.gx = (int16_t)(buffer[8] << 8 | buffer[9]) / 65.5;
        data.gy = (int16_t)(buffer[10] << 8 | buffer[11]) / 65.5;
        data.gz = (int16_t)(buffer[12] << 8 | buffer[13]) / 65.5;
        
        return data;
    }
};
```

## 6. 固件资源分配

### 6.1 内存映射

| 区域 | 大小 | 用途 |
|-----|------|------|
| Flash | 512KB (典型) | 程序代码 |
| SRAM | 128KB (典型) | 数据变量 |
| CCM RAM | 64KB | 核心数据 (快速访问) |

### 6.2 外设分配

| 外设 | 用途 |
|-----|------|
| TIM1/TIM8 | PWM 输出 (电机控制) |
| TIM2/TIM3 | 编码器接口 |
| USART1 | rosserial 通信 |
| I2C1 | MPU9250 IMU |
| GPIO | 控制/状态信号 |
