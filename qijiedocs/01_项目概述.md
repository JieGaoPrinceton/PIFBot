# PIFBot 项目概述

## 1. 项目简介

PIFBot 是一个基于 ROS（Robot Operating System）的移动机器人平台，主要用于运行 SLAM（同步定位与建图）算法。该机器人能够自主构建环境地图并在地图中导航移动。

## 2. 核心功能

| 功能模块 | 描述 |
|---------|------|
| SLAM 建图 | 支持实时环境地图构建 |
| 自主导航 | 基于建图结果的路径规划与导航 |
| 回环检测 | 支持多场景的回环检测功能 |
| 多会话 SLAM | 支持多次运行的数据融合 |

## 3. 技术栈概览

```
PIFBot/
├── firmware/          # 嵌入式固件 (STM32)
├── hardware/          # 硬件设计文件
├── mechanical/        # 机械结构设计 (FreeCAD)
├── ros/               # ROS 软件包
└── assets/            # 文档资源
```

### 3.1 软件技术栈

| 层级 | 技术 | 版本 |
|-----|------|-----|
| 操作系统 | Ubuntu | 16.04 Xenial |
| 机器人框架 | ROS | Kinetic |
| 嵌入式框架 | STM-IDF | 自定义框架 |
| 编程语言 | C/C++ | - |

### 3.2 硬件技术栈

| 组件 | 型号/规格 |

| 激光雷达 | RPLIDAR A1 |
| RGB-D 相机 | Intel RealSense D415 |
| 电机驱动 | TB6560 |
| 步进电机 | 42×42mm |
| IMU | MPU9250 (9-DOF) |

## 4. 机器人规格参数

| 参数 | 数值 |
|-----|------|
| 底盘直径 | 200 mm |
| 最大高度 | 212 mm |
| 轮子直径 | 65 mm |
| 轴距 | 165 mm |
| 驱动方式 | 差速驱动 |

## 5. 项目特点

1. **完全开源**: 所有硬件设计、固件代码、机械设计均在 MIT 许可证下开源
2. **模块化设计**: 各组件可独立替换和升级
3. **多传感器融合**: 集成激光雷达、RGB-D相机、里程计、IMU
4. **ROS 生态兼容**: 与 ROS 生态完全兼容，可使用各类 SLAM 和导航算法
